# Lumi — AIアシスタント搭載型習慣トラッキングアプリ

> 「無理のない範囲で習慣化できる」をコンセプトとし、**管理**ではなく**継続**に重きをおいたアプリ
> `アプリURL`: https://lumi-project-kappa.vercel.app/

## 開発したきっかけ

**58ハッカソン**（テーマ：「先端」）にsimaaoと2人チームで参加したことをきっかけに開発しました。

自分たち自身が「習慣化が苦手」という悩みを抱えており、既存のタスク・習慣管理アプリを試しても継続できないという経験がありました。既存アプリの多くは**タスクの「管理」に重点**を置いており、継続のモチベーション維持という観点が弱いと感じていました。

そこで Lumi は既存アプリとの差別化として、**「ユーザーが楽しく、無理のない範囲で習慣化できる」** をメインコンセプトに開発を進めました。

## プロジェクト概要

Lumi はペット育成と習慣管理を組み合わせたアプリです。習慣を達成するとペットキャラクター「Lumi」が成長・反応し、継続のモチベーションを高めます。また、ユーザーの生活習慣に基づいて習慣ごとに達成する重み**heaviness**を計算に加えて、Lumiがユーザーに最適されたアドバイスをする
---

## 現在の機能

### 認証
- メールアドレス・パスワードによるログイン・新規登録

### オンボーディング
初回ログイン時に3つの質問でユーザープロフィールを作成します。
1. 平均的なタスクの所要時間の選択
2. 集中力がピークになる時間帯の選択
3. 取り組みたい習慣タイプの優先順位設定（運動・学習・食事・睡眠・その他）

### ダッシュボード(メイン画面)
- 今日の習慣リストを表示し、チェックで完了記録ができる
- サイドパネルにペットキャラクター Lumi をリアルタイム表示(デスクトップ)
- 今日の達成タスク数を表示

### 習慣登録・編集
以下の項目を設定して習慣を登録できます。
- 習慣名
- 所要時間
- 実行する時間帯（朝・昼・夜・深夜）
- カテゴリ（運動・学習・食事・睡眠・その他）
- 目標日数（7 / 14 / 21 / 30 日）

### ペット育成システム
習慣の達成状況に応じて Lumi のステータスが変化します。

| ステータス | 説明 |
|---|---|
| 信頼値 | 習慣を達成すると上昇、未達成で低下 |
| 疲労度 | 1つの時間帯に習慣を詰め込みすぎると上昇 |
| 表情 | 信頼値・疲労度の組み合わせで joy / normal / sad に変化 |
| 親密レベル | 信頼値の蓄積によりレベルアップ・ダウン |

習慣ごとに達成する重みを設定することで、過負荷を検知して疲労度計算に反映しています。


### カレンダー（成長の軌跡）
- 月次カレンダービューで習慣の達成履歴を振り返れる
- 習慣タイプ別の円グラフ表示
- その日の達成済みタスク一覧

## 今後の追加予定機能

| 機能 | 概要 |
|---|---|
| AI 分析相談室 | ユーザーの行動ログを Gemini API で解析し、最適な習慣を提案する |
| Google 認証 | ソーシャルログインに対応し、登録の手間を削減する |
